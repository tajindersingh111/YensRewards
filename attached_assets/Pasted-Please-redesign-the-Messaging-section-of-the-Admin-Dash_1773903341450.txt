Please redesign the Messaging section of the Admin Dashboard to look more minimalist, modern, and professional, while still keeping the Yen’s brand identity.

Page to redesign:
- Admin Dashboard > Messages
- Sub-tabs: Send, History, Templates

Overall design direction:
- Clean admin dashboard style
- Minimalist and professional
- More white space
- Softer, more refined use of yellow as an accent colour only
- Strong visual hierarchy
- Premium but simple feel
- Easy for staff to use quickly
- Mobile responsive as well as desktop friendly

Layout requirements:
1. Keep the existing top navigation and structure, but improve the visual styling.
2. The Messages section should feel like a proper communications centre.
3. Replace the current large yellow blocks/buttons with a more elegant tab system:
   - Use white or very light backgrounds
   - Use yellow only for active states, hover accents, small highlights, or badges
4. Improve spacing and alignment throughout.
5. Reduce visual clutter and make the page feel lighter.

Templates page improvements:
1. Show each template in a clean card layout with:
   - Template name
   - Category/type
   - Delivery method badge (Email / SMS / Both)
   - Default badge if applicable
   - Short subject line preview
   - Small thumbnail preview of the template
   - Action icons: Edit, Delete, Preview, Set Default
2. Cards should have:
   - White background
   - Soft border
   - Subtle shadow
   - Rounded corners
   - Better internal padding
3. Cards should be arranged in a neat responsive grid.
4. Make the Add Template button more elegant and premium.

Typography:
- Use a clean sans serif font
- Improve font hierarchy between title, subtitle, labels, and metadata
- Make headings more modern and less bulky
- Use dark navy or charcoal for key text
- Use muted grey for secondary information

Colour usage:
- Main background: very light grey or off-white
- Card background: white
- Primary text: dark navy or charcoal
- Secondary text: muted grey
- Accent colour: Yen’s yellow, but only as an accent, not dominant everywhere
- Use yellow for active tabs, badges, buttons, and small highlights only

Specific UI improvements:
- Create a cleaner section header for “Message Templates”
- Add a search bar above templates
- Add filter dropdowns for:
  - Message type
  - Delivery channel
  - Default/non-default
- Add sorting options such as newest, oldest, alphabetical
- Improve icon consistency across the page
- Make all buttons and tabs feel more modern and less blocky

Template preview area:
- Preview image should be better contained inside the card
- Use a framed preview container with a light border
- Allow a “Quick Preview” modal when clicking View
- Keep the preview compact so the cards do not become too tall

Interaction design:
- Smooth hover effects on cards and buttons
- Clear active states on tabs
- Better visual feedback for edit/delete/default actions
- Keep interactions subtle and professional, not flashy

Style references:
- Think modern SaaS admin dashboard
- Clean like Stripe, Notion, Linear, or a polished CRM backend
- Minimal, elegant, efficient, and business-like

Technical request:
- Keep the page structure compatible with the existing Replit app
- Reuse existing data and functionality where possible
- Focus on front-end redesign and UI polish
- Do not remove existing functionality
- Ensure the design is responsive and accessible

Please produce:
1. Updated page layout
2. Improved template card design
3. Better tab styling
4. Search and filter bar
5. Cleaner action buttons and badges
6. Responsive behaviour for tablet and mobile
